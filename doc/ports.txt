Memory Map
------------------------------------------------------------------------
$00-$1F Регистры
$20-$3C Порты
$3D     SP[ 7:0]
$3D     SP[15:8]
$3F     SREG
------------------------------------------------------------------------
PORT
------------------------------------------------------------------------
$00     Банк памяти $F000-$FFFF (4 kb)
$01     -- старший банк --
$02     Скан-код клавиатуры
$03     Нажата ли клавиша (бит 0)
$04 RW  Курсор X (текстовый)
$05 RW  Курсор Y (текстовый)
$06     Таймер (LO)
$07     Таймер (HI) в миллисекундах
------------------------------------------------------------------------
$08 RW  SPI Data
$09 W   SPI Command: 0 Init, 1 Put/Get, 2 CE=0, 3 CE=1
    R   Bit 1 - Timeout
        Bit 0 - Busy
------------------------------------------------------------------------
$0A     Mouse.X [0..255]
$0B     Mouse.Y [0..255]
$0C     Mouse.C [2 бита]
$0E     Mouse.X.Hi [старшие биты]
------------------------------------------------------------------------
$0D     Видеорежим:

        0 - TEXTMODE
        1 - 320x200x8 64k
        2 - 256x240x2 16k
        3 - 320x200x4 32k
